# Etch-a-Sketch

A simple, interactive drawing application built with HTML, CSS, and JavaScript as part of the Foundations Course of The Odin Project. This project demonstrates core JavaScript concepts such as DOM manipulation, event handling, and dynamic element creation.

## Live Demo

🔗 [Live Preview](https://alexybnz.github.io/etch-a-sketch/)

## Repository

🔗 [Source Code](https://github.com/alexybnz/etch-a-sketch)

## Features

* Draw by hovering over grid squares
* Generate customizable grid sizes
* Clear and reset the drawing board
* Responsive design for different screen sizes
* Interactive user experience
* Real-time DOM updates
* Adjustable sketch area dimensions

## Built With

* HTML5
* CSS3
* JavaScript (ES6)

## What I Learned

Through this project, I practiced and improved my understanding of:

* JavaScript functions and scope

* Handling user input
* CSS Grid and Flexbox
* Writing clean and reusable code

## Challenges

One of the biggest challenges was dynamically creating and managing the sketch grid while maintaining performance and responsiveness, such as:

* Efficiently generating large numbers of grid squares
* Maintaining consistent square sizing across different grid dimensions
* Handling user input for custom grid sizes
* Removing and recreating grid elements without affecting performance
* Ensuring the layout remained responsive on different screen sizes
