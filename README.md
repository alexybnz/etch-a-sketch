Etch-a-Sketch

A simple, interactive drawing application built with HTML, CSS, and JavaScript as part of the Foundations Course of The Odin Project. This project recreates the classic Etch-a-Sketch experience while demonstrating core JavaScript concepts such as DOM manipulation, event handling, and dynamic element creation.

Dynamically generate grids of different sizes
Clear and reset the drawing board
Responsive design for various screen sizes
Smooth and interactive drawing experience
Customizable grid dimensions
Real-time DOM updates

Challenges

One of the biggest challenges was dynamically generating and managing the grid while ensuring the application remained responsive and efficient. Some specific challenges included:

Creating and removing large numbers of DOM elements efficiently
Maintaining consistent square dimensions regardless of grid size
Handling user input for custom grid sizes
Preventing performance issues with larger grids
Keeping the layout responsive across different screen sizes
